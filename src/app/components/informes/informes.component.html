REQUISITOS

Para adoptar a uno de nuestros perritos tienes que cubrir los siguentes requisitos:

  • Agendar una cita via telefonica, puedes llamar sin costo al número 018000726282.
  • Acordar fecha para entrevista.
  • Para la entrevista es necesario asista toda la familia que habita en el mismo domicilio.
  • Presentar documentos de identificación oficial.
  • Comprobante de domicilio reciente.
  • Presentar un donativo(efectivo o en especie), para seguir apoyando a más perritos.
  • Traer correa y caja para transportar perros de tamaño adecuado.
  • Aceptar terminos de cuidados.


PROCESO DE ADOPCIÓN

Para poder continuar con el proceso, debes haber cumplido con cada uno de los requisitos que se mencionaron anteriormente. Una vez entregada la documentación se evaluará que cumpla con las aspectativas para ser adoptante, se te otorgará un documento en el departamento correspondiente, en el cuál se validará la adopción del perrito. Ahora si puedes llevar a tu nuevo compañero a casa, recuerda cuidar y querer mucho a tu nueva mascota.
